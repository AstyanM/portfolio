## Introduction

Ceci est un projet d'exemple. Pour voir vos vrais projets :

1. Placez vos exports Notion dans le dossier `_raw_exports/`
2. Lancez `npm run ingest` pour les traiter
3. Les projets apparaitront automatiquement

## Fonctionnalités du portfolio

- Dark/Light mode
- Bilingue FR/EN
- Filtres par tags
- Animations fluides
- Images optimisées
