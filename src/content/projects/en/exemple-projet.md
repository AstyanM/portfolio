## Introduction

This is an example project. To see your real projects:

1. Place your Notion exports in the `_raw_exports/` folder
2. Run `npm run ingest` to process them
3. Projects will appear automatically

## Portfolio Features

- Dark/Light mode
- Bilingual FR/EN
- Tag filters
- Smooth animations
- Optimized images
